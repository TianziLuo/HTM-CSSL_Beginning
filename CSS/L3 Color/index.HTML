Lesson 3: Color

Summary:

  • Color Value:
      Predefined Color Name: eg.black/red/blue
      RGB: (0,0,0) - (255,255,255) = (red, green, and blue)
      RGBA: (0,0,0,0) - (255,255,255,1) = (red, green, blue, alpha) alpha for transparent.
      HEX: #RRGGBB
      HSL: (hue, saturation, lightness)
  • Background Color: background-color:lightyellow
  • Text Color: color:black
  • Border Color: border-color:tomato
