Lesson 2: Selector

Summary:

  • Universal Selector: most general. Reset the CSS.
  • Element Selector: more general.
  • Class Selector: more specific attribute. Select elements with a specific classname. Reusable.
  • Id Selector: most specific. Used to select one unique element within a page. Not recommand!
  • !Important: Add more importance to a property/value than normal. Not recommand!
