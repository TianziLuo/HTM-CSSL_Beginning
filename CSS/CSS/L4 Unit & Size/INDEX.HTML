CSS Units

Summary:

  • Lengths:
    •  Absolute length: not relative to anything else.
      Recommand for print layout
      Unit: px;cm;mm etc..
     Relative length: elative to size of the parent element's font, or the size of the viewport.
      Good for use on-screen.
      Unit: rem;em;vw;vh etc..
  • em & rem
    •  em Unit:The em unit allows setting the font size of an element relative to the font size of its parent.
      rem Unit: The rem is based upon the font-size value of the root element, which is the <html> element.
